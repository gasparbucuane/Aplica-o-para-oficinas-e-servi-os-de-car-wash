# Aplica-o-para-oficinas-e-servi-os-de-car-wash
A aplicação que permita adicionar clientes, carros, serviços, e gestão de preços.
Funcionalidades e Módulos Principais
1. Gestão de Clientes
Adicionar e Editar Clientes: Formulários para adicionar novos clientes e editar informações existentes (nome,id, e-mail).
Histórico de Serviços: Registro do histórico de serviços realizados para cada cliente.
2. Gestão de Carros
Adicionar e Editar Carros: Formulários para registrar informações do veículo (modelo, marca, placa, cliente proprietário).
3. Gestão de Serviços
Adicionar e Editar Serviços: Formulários para definir diferentes tipos de serviços (troca de óleo, alinhamento, lavagem completa, etc.).
Descrição dos Serviços: Detalhes de cada serviço, incluindo tempo estimado e materiais necessários.
4. Gestão de Preços
Definição de Preços: Interface para definir e atualizar preços dos serviços.
